<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* A web browser such as __Google Chrome__ or __Firefox__

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bIinno/jc2023coding.git
   ```
2. Navigate to the repository folder:
   ```sh
   cd jc2023coding
   ```
3.  Open the `index.html` file in your preferred web browser to view the website:
   ```sh
    open index.html
   ```

## Usage

* Open the `index.html` file in your preferred web browser.

* Interact with the website elements to explore the website's features and functionality.
